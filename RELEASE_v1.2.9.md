## BeatBlocks Control 1.2.9 — vector icons (no images)

### Changes
- **All overlay icons** are drawn with simple shapes (`DrawContext.fill`) — no PNG/SVG textures.
- **Unified size**: every icon renders at **14×14** px; active controls use Spotify green, inactive use dim gray.
- Removed `GuiIconRegistry` and icon texture upload path.

### Install
Replace your mod JAR and fully quit + restart Minecraft.
