Quick start — 3 steps

1
Spicetify + bridge

Copy beatblocks-api.js to Extensions, run spicetify apply, restart Spotify. Windows: scripts/setup-spicetify-bridge.ps1

2
Install the mod

Download the JAR for your exact MC version → mods/ + Fabric API.

3
Play

Spotify running → Minecraft → Alt+I or /sp. J / K / L for transport.
