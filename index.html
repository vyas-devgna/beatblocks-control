Quick start

1
Install Spicetify + the bridge

Copy beatblocks-api.js into Spicetify Extensions, run spicetify config extensions beatblocks-api.js and spicetify apply, then restart Spotify. Or use scripts/setup-spicetify-bridge.ps1 on Windows.

2
Add the mod

Download the JAR for your exact Minecraft version (GitHub Releases or Modrinth). Put it in mods/ with Fabric API.

3
Play

Start Spotify, launch Minecraft, press Alt+I for the overlay or /sp in chat. Use J / K / L for previous / play-pause / next.
