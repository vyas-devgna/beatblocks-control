BeatBlocks Control

Spotify in Minecraft — overlay, HUD, and hotkeys via a local Spicetify bridge.

Fabric mod Minecraft 1.21 – 1.21.11 Java 21+ No API keys
Download latest JARs View on GitHub

Quick start

1
Install Spicetify + extension

Copy beatblocks-api.js into Spicetify Extensions, run spicetify apply, restart Spotify.

2
Add the Fabric mod

Pick the JAR for your exact Minecraft version from Releases and put it in your instance mods folder with Fabric API.

3
Play

Open overlay with Alt+I (default) or /sp. Use J / K / L for prev / play-pause / next.

Features

  • Enhanced overlay — playlists, albums, liked songs, queue
  • Default overlay — settings, HUD scale, mode picker
  • Now-playing HUD with album art
  • Cached cover art from Spotify CDN
  • Local bridge on 127.0.0.1 — credentials stay in Spotify

Supported versions

One JAR per patch release — do not mix versions.
