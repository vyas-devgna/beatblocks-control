Free Fabric mod

BeatBlocks Control

Control Spotify from inside Minecraft — without leaving your world.

In one sentence

A Minecraft mod that shows Spotify on your screen and lets you skip songs with hotkeys, using a small helper on your PC (not the cloud).

You will need
  • Minecraft Java 1.21 – 1.21.11 + Fabric + Fabric API
  • Spotify desktop app (open while you play)
  • Spicetify + beatblocks-api.js from the repo
No API keys Stays on your PC Client-only mod
Download mod Modrinth GitHub

Setup

3 steps to get started

Minecraft → your PC → Spotify
1
Set up Spotify helper

Install Spicetify, copy beatblocks-api.js to the Extensions folder, run spicetify apply, then restart Spotify.

2
Install the mod

Download the JAR that exactly matches your Minecraft version. Put it in mods/ with Fabric API.

3
Play

Start a song in Spotify, launch Minecraft, press Alt+I to open the overlay (or type /sp).

Which JAR do I download?

Pick the file that matches your Minecraft version exactly — for example beatblocks-control-mc-1.21.5.jar for Minecraft 1.21.5.
